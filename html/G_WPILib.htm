WPILib Namespaces
The WPILib Root
Namespaces
NamespaceDescription
WPILib
The WPILib namespace contains all of the component classes for the library.
WPILib.Buttons
The Buttons namespace contains Buttons for use with Commands.
WPILib.CAN
The CAN namespace contains the exception filtering for interfacing with CAN Jaguars.
WPILib.Commands
The Commands namespace contains classes for creating Commands and Subsystems.
WPILib.Exceptions
The Exceptions namespace contains custom exceptions for the WPILib.
WPILib.Extras
The Extras namespace contains classes not included in the officially WPILib but we think they should be.
WPILib.Interfaces
The Interfaces namespace contains common interfaces used by the WPILib.
WPILib.Internal
The Internal namespace contains class implementations for the RoboRIO.
WPILib.LiveWindows
The LiveWindows namespace contains LiveWindow components. Note that in java this would be called LiveWindow, but C# doesnt allow a namespace to match a class.
WPILib.SmartDashboards
The SmartDashboard namespace contains classes for interfacing with the Smart Dashboard.
