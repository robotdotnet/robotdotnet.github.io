Projects

Here you will find a list of all projects for WPILib

  • WPILib WPILib native in .NET
  • WPILib-CTRE Libraries for CANTalonSRX and other CTRE devices
  • NetworkTables FRC NetworkTables both in native .NET and ntcore wrapper
  • CameraServer Wrapper around the WPILib cscore library for interfacing with CameraServer

  • FRC-OpenCvSharp OpenCV for .NET. Fork with custom builds for the WPILib build system

  • FRC Visual Studio Extension Our extension for integrating the build and deploy process straight into Visual Studio. Also includes templates for easy project creation.
