Remarks
The Command will be started after all the previously added commands.

Once the Command is started, it will be run until it finishes or the time expires, whichever is sooner. Note that the give Command will have no knowledge that it is on a timer.

Note that any requirements thee given Command has will be added to the group. For this reason, a Command's requirements cannot be changed after being added to a group.

It is recommended that this method be called in the constructor.
