Implementing a Shooter Wheel Mechanism

A shooter wheel mechanism is one of the most common mechanisms used in FRC, and is a great candidate for simulation. However, it is one of the hardest as it is a dynamic system. In order to simulate it properly, we must know 2 things. First, we must know the inertia of the system. Secondly we must know the drag on the system that causes it to deaccelerate when no power is applied.
