Remarks
Commands in a CommandGroup are added using the AddSequential(Command) method and are called sequentially. Command Groups are themselves commands and can be given to other Command Groups.

Command Groups will carry all of the requirements of their subcommands Additional requirements can be specified by calling Requires(Subsystem) normally in the constructor.

Command Groups can also execute commands in parallel, simply by adding them using AddParallel(Command).
