Remarks
You can build a simple robot program off of this by overrriding the RobotInit, Disabled, Autonomous, and OperatorControl methods. The StartCompetition method will call these methods (sometimes repeatedly) depending on the state of the competition.

Alternatively, you can override the RobotMain method and manage all aspects of the robot yourself.
