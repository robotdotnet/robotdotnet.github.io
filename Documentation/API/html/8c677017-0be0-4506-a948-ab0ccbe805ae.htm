Running the MonoGame Simulator

Once the MonoGame simultor project has been created, it is easy to get it running. As long is it is the startup project, whenever you click the Start button in Visual Studio it will run. Note that the robot project does NOT need to be the startup project in order for it to be deployed properly to the robot. The extension will automatically find the robot project and deploy it.

Controlling the Driver Station

The MonoGame simulator includes a custom DriverStation for controlling your robot code. This is similar to the offical DriverStation for enabling and Disabling the robot, however joystick are different.

On the right side of the window, you will see 4 control boxes. These allow you to select which joysticks is sent to the robot.
