Namespaces
NamespaceDescription
WPILib
Contains all of the base component classes for the WPILib.
WPILib.Buttons
Contains Button APIs for use with Commands.
WPILib.CAN
Contains the exception filtering for interfacing with CAN Jaguars.
WPILib.Commands
Contains the Command API for creating commands and subsystems.
WPILib.Exceptions
Contains custom exceptions for the WPILib.
WPILib.Extras
Contsins classes not included in the official WPILib but still very useful.
WPILib.Interfaces
Contains the interfaces used by the WPILib.
WPILib.Internal
Contains RoboRIO specific class implementations.
WPILib.LiveWindows
Contains components for interfacing with the SmartDashboard LiveWindows.
WPILib.SmartDashboards
Contains the APIs for communicating with the SmartDashboard.
WPILib.Vision
Contains the APIs for interfacing with NIVision.
