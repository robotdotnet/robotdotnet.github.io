Remarks
Quadrature encoders are devices that count shaft rotation and can sense direction. The output of the QuadEncoder class is an integer that can count either up or down, and can go negative for reverse direction counting. When creating Quad Encoders, a direction is supplied that changes the sense of the output to make code more readable if the encoder is mounter such that forwrd movement generates negative values. Quadrature encoders have two digital outputs, a A channel and a B channel that are out of phase with each other to allow the FPGA to do direction sensing.

All encoders will immediately start counting - Reset them if you need them to be zeroed before use.
