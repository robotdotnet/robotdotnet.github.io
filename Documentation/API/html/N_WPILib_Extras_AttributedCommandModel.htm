Contains the AttributedCommandModel for easy Command based robot code.
Classes
  ClassDescription
AttributedRobot
The AttributedRobot class. Derive from this class to use the Attributed Command Model
ExportSubsystemAttribute
Apply this attribute to subsystems to have AttributedRobot auto-load it.
ImportSubsystemAttribute
Apply this attribute to a subsystem parameter in a Command constructor to disambiguate between multiple-exported subsystems of the same type for the AttributedRobot auto-loading.
RunCommandAtPhaseStartAttribute
Apply this attribute to a command to have it started when the respective match phase starts.
RunCommandAttribute
Apply subclasses of this attribute to commands to tie them to triggering conditions.
RunCommandOnJoystickAttribute
Run a command on a JoystickButton event.
RunCommandOnNetworkKeyAttribute
Run a command on a NetworkButton event.
