Installing the Runtime onto the RoboRIO

This will guide you through installing Mono on the RoboRIO. Please perform these steps either USB or Ethernet connected to the RoboRIO. Wireless is not recommended. Note that you need to make sure you have imaged your RoboRIO following FIRST's instructions, which can be found here.

Visual Studio Online Installation - Recommended

If you do not already have Visual Studio and the FRC extension installed, please follow the instructions here. Then follow the steps below.

  1. Open Visual Studio. If you have not already set your team number, please do so by going to FRC | Settings.

  2. Go to FRC | Mono Tools and select Download Mono. This will download the proper version of Mono to your system. A popup will appear after the download has finished asking if you would like to install Mono. Click Yes.

  3. Mono will now be getting installed to the RoboRIO. Once it is finished, a popup will appear indicating if the install was successful or not.

  4. If successful, Mono will now be properly installed on your RoboRIO. See creating your first robot project for instructions on how to create a project and deploy it to the RoboRIO.

Offline Installation

If you do not already have Visual Studio and the FRC extension installed, please follow the instructions here. Then follow the steps below.

  1. You can download the Mono runtime on another system using the download link here. Download Mono 4.2.1 Then copy this zip file to the system connected to the RoboRIO, and follow the steps below.

  2. Open Visual Studio. If you have not already set your team number, please do so by going to FRC | Settings.

  3. Go to FRC | Mono Tools and select Install Mono. If Mono has not been downloaded to the current system, it will ask if you would like to look for it. Click yes, and then search for the downloaded file.

  4. Mono will now be getting installed to the RoboRIO. Once it is finished, a popup will appear indicating if the install was successful or not.

  5. If successful, Mono will now be properly installed on your RoboRIO. See creating your first robot project for instructions on how to create a project and deploy it to the RoboRIO.
