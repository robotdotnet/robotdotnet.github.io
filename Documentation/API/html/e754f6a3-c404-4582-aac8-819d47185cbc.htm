Creating a Mono Game Simulator

[This is preliminary documentation and is subject to change.]

MonoGame is the recommended front end for creating simulators. Below are the steps to get the basic simulator working.

Installing MonoGame and creating a MonoGame Project

  1. Install the latest MonoGame release for your operating system. Follow the instructions at the link to get it installed. The current release is 3.4.

  2. Open your solution with your main robot project.

  3. Right click on your solution file in Visual Studio, and select Add | New Project.

  4. In the Visual C# list, select FRC, then select the MonoGame Simulator. Enter a name for your simulator and click OK.

  5. Your solution should now have 2 projects in it, with the simulator set as the startup project. Note that this is correct, and you do not need to have the main robot project as the startup project in order for it to deploy correctly. See Running the MonoGame Simulator for instructions on how to run it, and individual mechanism pages for specifics.
