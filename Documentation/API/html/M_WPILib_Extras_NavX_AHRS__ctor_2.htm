The Update Rate (Hz) [4..60] (Default 50)
Remarks
The update rate may be between 4 Hz and 60 Hz, representing the number of updates per second sent by the sensor.

This constructor should be used if communicating via SPI.

Note that increasing the update rate may increase the CPU utilization.
