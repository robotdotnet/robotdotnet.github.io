RobotInit()

Override this method to add additional code that executes after WPILib.Extras.AttributedCommandModel.AttributedRobot._RobotInit.

Declaration
public virtual void RobotInit()
Remarks

Robot-wide initialization code should go here. Users should override this method for default Robot-wide initialiation which will be called when the robot is first powered on. It will be called exactly one time.

Warning: The Driver Station "Robot Code" light and FMS "Robot Ready" indicators will be off until RobotInit() exits. Code in RobotInit() that waits for enable will cause the robot to never indicate that the code is ready, causing the robot to be bypassed in a match.
