Remarks

Analog potentiometers read in an analog voltage that corresponds

to a position.The position is

in whichever units you choose, by way of the scaling and offset

constants passed to the constructor.

AnalogPotentiometer(Int32, Double, Double)

Creates a new Analog Potentiometer on the specified channel. [0..3] on RIO, [4..7] on MXP.

Declaration
public AnalogPotentiometer(int channel, double fullRange = 1, double offset = 0)
Parameters
Type Name Description
Int32 channel

The channel this potentiometer is plugged into.
Double fullRange

The scaling to multiply the fraction by to get a meaningful unit.
Double offset

The offset to add to the scaled value for controlling the zero value.
Remarks

Use the fullRange and offset values so that the output produces meaningful values. I.E: you have a 270 degree potentiometer and you want the output to be degrees with the halfway point at 0 degrees. The fullRange value is 270.0(degrees) and the offset is -135.0 since the halfway point after scaling is 135 degrees.

This will calculate the result from the fullRange times the fraction of the supply voltage, plus the offset.
