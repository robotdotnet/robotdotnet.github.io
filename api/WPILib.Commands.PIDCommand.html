Syntax
public abstract class PIDCommand : Command, INamedSendable, ISendable, ITableListener, IPIDSource, IPIDOutput
Remarks

It provides some conveniance methods to run an internal PIDController. It will also start and stop said PIDController when the PIDCommand is first initialized and ended/interrupted.
