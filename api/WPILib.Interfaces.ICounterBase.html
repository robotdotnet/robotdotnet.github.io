MaxPeriod

Sets the maximum period for stopped detection, in seconds.

Declaration
double MaxPeriod { set; }
Property Value
Type Description
Double
Remarks

Sets the value that represents the maximum period of the Encoder before it will assume that the attached device is stopped. This timeout allows users to determine if the wheels or of ther shaft has stopped rotating. This method compensates for the decoding type.
