Class SendableChooser

The SendableChooser class is a useful tool for presenting a selection of options to the SmartDashboard.

Remarks

One use for this is to be able to select between multiple autonomous modes. You can do this by putting every possible Command you want to run as an autonomous into a SendableChooser and then put it into the SmartDashboard to have a list of options appear on the laptop. Once autonomous starts, simply ask the SendableChooser what the selected value is.
