GetMatchTime()

Return the approximate match time since the beginning of autonomous

Declaration
public static double GetMatchTime()
Returns
Type Description
Double

Match time since the beginning of autonomous.
Remarks

The FMS does not currently send the official match time to the robots This returns the time since the enable signal sent from the Driver Station At the beginning of autonomous, the time is reset to 0.0 seconds At the beginning of teleop, the time is reset to +15.0 seconds If the robot is disabled, this returns 0.0 seconds

Warning: This is not an official time (so it cannot be used to argue with referees)
