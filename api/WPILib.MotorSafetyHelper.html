Class MotorSafetyHelper

The MotorSafetyHelper object is constructed for every object that wants to implement the Motor Safety protocol.

Inheritance
Object
MotorSafetyHelper
Namespace:WPILib
Assembly:WPILib.dll
Syntax
public class MotorSafetyHelper : object
Remarks

The helper object has the code to actually do the timing and call the motors Stop() method when the timeout expires. The motor object is expected to call the Feed() method whenever the motors value is updated.

Remarks
