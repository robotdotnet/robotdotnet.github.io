ReadBufferSize

Sets the size of the input buffer.

Declaration
public int ReadBufferSize { set; }
Property Value
Type Description
Int32
Remarks

Specify the amount of data that can be stored before data from the device is returned to Read(Int32). If you want data that is received immediately, set this to 1.

If the buffer is not filled before the read timeout expired, all data that has been received so far will be returned.

WriteBufferMode

Sets the flushing behavior of the output buffer

Declaration
public SerialPort.WriteBufferModeEnum WriteBufferMode { set; }
Property Value
Type Description
SerialPort.WriteBufferModeEnum
Remarks

When set to FlushOnAccess, data is synchronously written to the serial port after each call to Write(Byte[], Int32)

When set to FlushWhenFull, data is only written to the serial port when the buffer is full or Flush() is called.
