Class PIDSubsystem

This class is designed to handle the case where there is a Subsystem which uses a single PIDController almost constantly.

Returns
Type Description
Double

The value the PID Loop should use as input.
Remarks

It returns the input for the PID loop, so if this command was based off of a gyro, then it shoudl return the angle of the gyro.

This method will be called in a different thread then the Scheduler thread.
