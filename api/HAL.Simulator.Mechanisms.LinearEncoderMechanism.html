The CPR of the encoder. If not a 1:1 ratio on the axle, scale this beforehand.
DCMotor model

The transmission model to use.
Double spoolRadius

The radius of your spool in Meters. (Use the radius of the up spool if using a cascade elevator).

The start height of your elevator relative to the reset sensor in meters. If no reset sensor then use 0.
