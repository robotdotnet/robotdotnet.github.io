Class IndependentNtCore

This class contains all NtCore methods exposed by the underlying library, running in an independent state

Remarks

The static NtCore, NetworkTable and RemoteProcedureCall all run using the same backend library. This means you cannot have both a client and a server running in the same user program. The IndependentNtCore, IndependentNetworkTable and IndependentRemoteProcedureCall get around this restriction, and allow multiple clients and servers in the same user program. Note that this is not supported by NetworkTables.Core.
